# GLOW-SG13G2

GLOW-SG13G2 (Gate Library for Open Flow) is a library of CMOS digital standard cells for [IHP SG13G2](https://github.com/IHP-GmbH/IHP-Open-PDK) process. 

The primary objective of this project is to design a rich library of standard digital cells that can be used in open source digital tools.
As a by-product it also provides a work flow that can be used to design new standard cells for IHP SG13G2 process.

An effort has been made to make the work flow process-agnostic as much as possible, so that it could be used with other PDKs that support open source tools.
